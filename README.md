# Earth wallpaper
Renders clouds, surface elevation and water reflection.  
The map changes according to the current season.  
The image is continuously updated depending on the angle of attack of the sun.

---

### Dependencies
+ `xplanet`: renders planets and other space-related stuff
+ [`nitrogen`](https://github.com/l3ib/nitrogen): standalone wallpaper setter (part of my [BashGNOME project](https://github.com/Tumlinh/BashGNOME))
